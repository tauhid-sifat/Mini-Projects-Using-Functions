# Mini-Projects Using Functions

This repository contains a collection of small Python projects designed to practice and demonstrate the use of functions. Each project focuses on a specific concept or application.

## Projects

1. Calculator
2. Number Guessing Game
3. Simple Password Generator
4. Text-Based Adventure Game
5. Unit Converter
6. Hangman Game
7. Expense Tracker
8. Recipe Manager
9. Flashcard Quizzer
10. Basic Quiz Application
11. Personal Journal
12. Sudoku Solver
13. Weather Information Fetcher
14. Tic-Tac-Toe Game
15. Markdown to HTML Converter
16. Text File Analyzer

## How to Use

Each project is contained in its own Python file. To run a project, navigate to its directory and run the Python script.


## Requirements

- Python 3.x
